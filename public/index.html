❓ Aide – Comment utiliser l'application

👤 Première connexion et accès

  • Si vous avez reçu une invitation par e-mail, ouvrez le lien reçu puis créez votre mot de passe pour activer votre accès.
  • Ensuite, saisissez votre adresse e-mail et votre mot de passe, puis cliquez sur Se connecter.
  • En cas d'oubli, cliquez sur Mot de passe oublié pour recevoir un lien de réinitialisation par e-mail.
  • Une fois connecté, vous accédez directement à votre propre fiche mensuelle d'entraîneur.

📅 Renseigner votre mois

  • Choisissez le mois souhaité avec le champ Mois.
  • Cliquez sur un jour dans le calendrier pour ouvrir la fenêtre de saisie.
  • Indiquez le nombre d'heures d'entraînement (par tranches de 0,25 h).
  • Cochez Jour de compétition si vous avez participé à une compétition ce jour-là, puis renseignez les détails du trajet : kilométrage, départ, arrivée, description, péage éventuel et frais d'hôtel si nécessaire.
  • Vous pouvez aussi saisir un achat pour le club (matériel, fournitures, etc.) sur la journée concernée.
  • Les justificatifs de péage, hôtel et achat peuvent être joints en PDF, JPG ou PNG. Ils sont obligatoires pour exporter la note de frais si un montant correspondant est saisi.
  • Cliquez sur Sauvegarder pour enregistrer la journée, ou sur Supprimer pour effacer les données du jour.

🔒 Fiche gelée

  • Si un bandeau Cette fiche est gelée apparaît, cela signifie que le mois a été verrouillé par l'administration.
  • Dans ce cas, vous pouvez consulter les données du mois, mais vous ne pouvez plus les modifier.
  • Si une correction est nécessaire, contactez un administrateur du club.

🎨 Code couleur du calendrier

  • Gris – Week-end
  • Rose/Rouge – Jour férié
  • Orange clair – Vacances scolaires
  • Vert – Entraînement saisi
  • Bleu – Compétition saisie
  • Violet – Achat saisi

📊 Récapitulatif mensuel

Le panneau sous le calendrier affiche automatiquement le total des heures, des jours de compétition, des kilomètres parcourus ainsi qu'une estimation du montant calculé pour le mois sélectionné.

Le remboursement kilométrique est calculé automatiquement selon le barème légal applicable aux voitures, en fonction de la puissance fiscale enregistrée et du cumul annuel des kilomètres.

📤 Exports

  • Exporter relevé d'heures – génère un document imprimable qui inclut les heures d'entraînement et les indemnités forfaitaires des jours de compétition.
  • Exporter note de frais – génère un document HTML imprimable récapitulant les remboursements kilométriques, les péages, les frais d'hôtel et les achats du mois (utilisez la fonction d'impression du navigateur pour le convertir en PDF).
  • Exporter backup JSON – enregistre une sauvegarde de vos données du mois dans un fichier JSON.

💡 Bonnes pratiques

  • Saisissez vos journées régulièrement pour éviter les oublis en fin de mois.
  • Conservez vos justificatifs de péage et d'hôtel et ajoutez-les directement sur les jours concernés.
  • Avant la fin du mois, vérifiez le récapitulatif et exportez vos documents si besoin.

🔧 Fonctions administrateur

  • + Ajout utilisateur – ajoute un nouveau profil puis permet de choisir le type Entraîneur, Bénévole ou Administrateur dans la fiche.
  • ✏️ Modifier – modifie le profil sélectionné et son type si nécessaire.
  • 🔒 Geler la fiche – verrouille ou déverrouille un mois pour empêcher les modifications.
  • 📥 Importer JSON – permet de réinjecter une sauvegarde JSON dans la fiche affichée.
  • ✉️ Inviter admin – envoie une invitation à un nouvel administrateur.
  • 📊 Voir synthèse du mois – ouvre un aperçu dans l'application pour le mois sélectionné, avec les heures payées, le nombre de compétitions, le salaire calculé et les indemnités kilométriques par profil, puis permet l'export CSV ou JSON.
  • Les administrateurs peuvent consulter et exporter les données de n'importe quel profil.
