❓ Aide – Comment utiliser l'application

👤 Connexion et inscription

  • Saisissez votre adresse e-mail et votre mot de passe, puis cliquez sur Se connecter.
  • Si vous n'avez pas encore de compte, cliquez sur S'inscrire pour en créer un.
  • En cas d'oubli de mot de passe, cliquez sur Mot de passe oublié pour recevoir un lien de réinitialisation par e-mail.

📅 Saisir une journée

  • Sélectionnez votre nom dans la liste Entraîneur et choisissez le mois souhaité.
  • Cliquez sur un jour dans le calendrier pour ouvrir la fenêtre de saisie.
  • Indiquez le nombre d'heures d'entraînement (par tranches de 0,5 h).
  • Cochez Jour de compétition si vous avez participé à une compétition ce jour-là, puis renseignez les détails du trajet (km, lieu de départ, lieu d'arrivée, description, péage).
  • Vous pouvez joindre un justificatif de péage (PDF, JPG ou PNG).
  • Cliquez sur Sauvegarder pour enregistrer l'entrée, ou sur Supprimer pour effacer les données du jour.

🎨 Code couleur du calendrier

  • Gris – Week-end
  • Rose/Rouge – Jour férié
  • Orange clair – Vacances scolaires
  • Vert – Entraînement saisi
  • Bleu – Compétition saisie

📊 Récapitulatif mensuel

Le panneau sous le calendrier affiche automatiquement le total des heures, des jours de compétition, des kilomètres parcourus et le montant total à payer pour le mois sélectionné.

📤 Exports

  • Exporter salaire (CSV) – génère un fichier tableur avec le détail des jours et le calcul des rémunérations.
  • Exporter note de frais km – génère un document HTML imprimable récapitulant les trajets de compétition (utilisez la fonction d'impression du navigateur pour le convertir en PDF).

📥 Import JSON

Vous pouvez importer les données d'un mois précédent depuis un fichier JSON de sauvegarde via le bouton Importer JSON.

🔧 Fonctions administrateur

  • + Entraîneur – ajoute un nouveau profil d'entraîneur avec ses tarifs.
  • ✏️ Modifier – modifie le profil et les tarifs de l'entraîneur sélectionné.
  • Les administrateurs peuvent consulter et exporter les données de n'importe quel entraîneur.
